# rs232-vi-sequence-controller (三段式電壓/電流時序控制器)

這是一個基於 Python 與 Tkinter 開發的圖形化介面工具，主要用於透過 RS232 控制可程式化直流電源供應器（Programmable DC Power Supply）。它允許使用者定義三個階段的電壓、電流以及持續時間，並能平滑地進行階段間的斜坡過渡（Ramp），最後以指定的次數重複循環。

## 功能特色

- **三段式設定**：可獨立設定階段 1、階段 2、階段 3 的目標電壓 (V)、電流 (A) 與持續時間 (ms)。
- **線性轉換 (Ramping)**：支援設定段落間（1→2, 2→3, 3→1）的轉換時間。若時間大於 50ms，程式會自動產生平滑的階梯式過渡；小於 50ms 則瞬間跳躍。
- **SCPI 指令通訊**：使用標準儀器控制指令（如 `:SOUR:VOLT`, `:SOUR:CURR`, `:OUTP ON/OFF`），相容於多數支援 SCPI 的電源供應器。
- **緊急停止功能**：隨時可以中斷循環，並將電壓電流歸零、強制關閉電源輸出，確保實驗安全。
- **非同步執行**：控制迴圈運行於獨立執行緒，確保執行期間 GUI 不會凍結。

## 環境需求

請確保您的系統已安裝 Python 3.x，並安裝以下依賴套件：

```bash
pip install pyserial numpy
```

## 使用說明

1. **硬體連接**：將您的電源供應器透過 RS232 傳輸線連接至電腦。
2. **確認通訊埠 (COM Port)**：
   請開啟 `RS232.py` 原始碼，將第 7 行的 `SERIAL_PORT` 修改為您的實際連接埠（例如 `COM3` 或 `/dev/ttyUSB0`）。並確認 `BAUDRATE` (預設 115200) 與硬體設定一致。
3. **啟動程式**：
   ```bash
   python RS232.py
   ```
4. **操作介面**：
   - 輸入各階段所需的電壓、電流與時間參數。
   - 設定循環次數。
   - 點擊「▶️ 開始循環」執行。
   - 若遇突發狀況，可隨時點選「⛔ 緊急停止」。
